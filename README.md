# Samaritans NGO – Volunteer & Event Management System

A full-stack NGO management platform developed for **Samaritans for the Nation** to streamline volunteer management, event tracking, resource allocation, and organizational activities through a centralized web application.

---

## Project Overview

The project transforms a static informational website into a dynamic and scalable web application with real-time data handling and efficient management capabilities.

The system helps NGOs:
- Manage volunteers and events
- Track resources and activities
- Improve coordination and accountability
- Reduce manual work and delays
- Maintain centralized data records

---

## Features

- Volunteer Registration & Management
- Event Management System
- Admin Dashboard
- Resource & Location Tracking
- Real-Time Data Handling
- CRUD Operations
- Authentication System
- Dynamic Web Interface
- Database Integration
- Scalable Full-Stack Architecture
